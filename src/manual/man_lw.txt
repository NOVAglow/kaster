Log Writer
  Description:

    The log writer is a Kaster's component that has the full privilege
    to access the Kaster's log file
    (whose path is '/usr/share/kaster/log.dat' by default).


  Options:

        --lw     : Call log writer
    -h, --help   : Print this manual page and exit
        --create : Create a log file
        --append : Append a string to the log file
        --log    : Just like --append, but append the string
                   with the date and time in front
        --clear  : Clear content in the log file
        --delete : Delete the log file


  Examples:

    1. Prints this manual page:
      ./kaster.py --lw
    2. Create a log file:
      ./kaster.py --lw --create
    3. Append a string to the log file:
      ./kaster.py --lw --append="foo"
    4. Append a string to the log file with date and time in front:
      ./kaster.py --lw --log="bar"
     (which prints something like "[15:03:50 10/09/2018] bar" to the log file)
    5. Clear the content in the log file:
      ./kaster.py --lw --clear
    6. Delete the log file:
      ./kaster.py --lw --delete
